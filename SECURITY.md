# Security Policy

## Supported Versions

The following versions of @nodebug/axios are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an email to nodebug@outlook.com. All security vulnerabilities will be promptly addressed.

Please do not disclose security-related issues publicly until a fix has been released.

## Security Best Practices

- Keep dependencies up to date
- Review code changes before merging
- Use HTTPS connections when possible
- Validate and sanitize all inputs

## Security Updates

Security updates are released as patch versions. We recommend keeping your @nodebug/axios package up to date with the latest patch version to ensure you have the latest security fixes.
